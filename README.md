## ‚úÖ First-Time Setup
Before running a script, you need to grant it execution permission **once**:

```markdown

```

Also you can add shortcut

for bash:
```markdown

source ~/.bashrc
```
for zsh:
```markdown
echo 'alias clean_branches="./clean_branches.sh"' >> ~/.zshrc
source ~/.zshrc
```
